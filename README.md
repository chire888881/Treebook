# Treebook
7/23討論結果  
Ａ.樹功能：  
1.新增root節點  
2.新增新節點（往上）     
3.新增平行節點（往左）
3-1.新增新節點（往右）  
4.每個節點可以以加上無限個tag  
5.每個節點可以加上一個敘述，沒有字數限制  
6.刪除新節點  
7.刪除根節點  
8.新增節點時間
